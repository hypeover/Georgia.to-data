🇬🇪 # Georgia Travel Data Scraper & Map Planner
While planning a trip to Georgia, I came across the website Georgia.to, which features a huge collection of beautiful places worth visiting. The problem? There are simply too many locations to easily plan a trip without getting overwhelmed or accidentally duplicating spots. Each object contains: latitude, longitude, link to thumbnail, link to page about the place, place's types and title of the place.

Each object includes: latitude, longitude, a thumbnail URL, a link to the place’s page, its types, and its title.

You can download the pre-generated data.json file or generate it yourself by running the Python script.
